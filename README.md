# GridLab YD

![GridLab YD Banner](./assets/banner.png)

## About

**GridLab YD** is an open-source creative composition tool designed to help photographers, videographers, designers, and creators organize and plan visual layouts using customizable grid systems and guides.

The project aims to provide creators with a simple and flexible way to improve framing, composition, and visual planning.

## Features

* Customizable composition grids
* Visual layout guides
* Multiple aspect ratio support
* Creative planning tools
* Lightweight and accessible interface
* Open-source and community-driven development

## Why GridLab YD?

Composition is one of the most important parts of visual storytelling. GridLab YD was created to help creators experiment with different layouts, improve their shots, and develop stronger visual awareness.
